CSCI-02: Gemini Core Sequence Diagram

        sequenceDiagram
            participant CLI as CSCI-01: CLI
            participant Core as CSCI-02: Core
            participant Tools as CSCI-03: Tools
            participant API as External: Gemini API

            CLI->>Core: INT-01: Start Agent with User Prompt
            activate Core

            Core->>Core: CSC-02.01: Build Prompt
            Core->>API: INT-03: Send Prompt for inference
            activate API

            API-->>Core: INT-03: Respond with Tool Call request
            deactivate API

            Core->>Core: CSC-02.02: Parse Tool Call
            Core->>Tools: INT-02: Execute Tool (e.g., readFile)
            activate Tools
            
            Note over Tools: Interacts with OS/Web (INT-05)
            Tools-->>Core: INT-02: Return Tool Result
            deactivate Tools

            Core->>API: INT-03: Send Tool Result for next inference
            activate API

            API-->>Core: INT-03: Respond with Final Answer
            deactivate API

            Core-->>CLI: Return Final Answer
            deactivate Core
